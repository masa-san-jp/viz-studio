# Viz Studio — データに、雰囲気を与える

CSV / JSON のデータを貼り付けると、three.js の3Dビジュアライズを生成し、
「気分のつまみ」を動かすだけで全体の雰囲気をその場で調整できる、単一ファイルのWebアプリ。

## つかいかた
- ブラウザで `index.html` を開くだけ（ビルド不要・外部依存なし）。
- 「データと列」を開いて CSV か JSON配列（1行目=見出し）を貼り「反映」。列を X／高さ／奥行き／色 に割り当て。
- **粒 / 柱** を選ぶ。ドラッグで回転・ホイールで拡大。

## 気分のつまみ（1本で複数のパラメータが同時に動く）
- **温度** — 寒色・静謐 ⇄ 暖色・熱
- **ドラマ** — 穏やか・平坦 ⇄ 劇的・そそり立つ（高さ・発光・コントラスト）
- **エネルギー** — 静寂 ⇄ 躍動・鼓動（回転・脈動）
- **空気感** — 澄んだ夜 ⇄ 霧・幻想（フォグ・空気）
- **密度感** — 繊細・ひとつぶ ⇄ 濃密・光の塊

## 技術
three.js（r140・`three.min.js` 同梱）。色を選ぶような原始的なUIでなく、感情・雰囲気を直感で調整するUXを志向。
